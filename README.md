# 🐍 Classic Snake Game (Python Final Project)

這是一個使用 Python 和 **Pygame** 函式庫開發的經典貪吃蛇遊戲。此專案實作了完整的遊戲迴圈，並採用物件導向 (OOP) 的方式設計，將角色行為與遊戲邏輯清晰分離。

## 📂 程式架構 (Project Structure)

本專案主要由三個核心部分組成：

### 1. `class Snake` (角色物件)
負責管理「蛇」的實體狀態與行為：
* **座標管理**：使用 List 儲存身體每一節的網格座標 `(x, y)`。
* **移動邏輯**：處理方向改變與座標更新，實現蛇的移動效果。
* **長度控制**：包含 `shrink()` 方法，配合遊戲邏輯來決定是否保留尾巴（變長）。

### 2. `class Game` (遊戲邏輯)
負責管理整體的規則判定與環境：
* **碰撞檢測**：判斷是否撞牆 (`Hit Wall`) 或撞到自身 (`Hit Self`)。
* **食物機制**：隨機生成食物並確保不與蛇身重疊。
* **狀態更新**：統一處理 `update()` 與畫面繪製 `draw()`。

### 3. `main()` (主程式)
程式的入口點，負責驅動遊戲引擎：
* **初始化**：設定 Pygame 環境、視窗大小與 FPS (每秒幀數)。
* **事件監聽**：捕捉鍵盤輸入 (上下左右) 並傳遞給 Snake 物件。
* **遊戲迴圈**：維持「輸入 -> 更新 -> 繪圖」的循環運作。
