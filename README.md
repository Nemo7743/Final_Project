# 🐍 Python 貪食蛇遊戲

這是一個使用 Python `pygame` 開發的進階貪食蛇遊戲。本專案展示了良好的模組化程式設計，將菜單、遊戲迴圈與邏輯規則分離。遊戲包含經典單人模式以及本地雙人對戰模式。

## ✨ 專案特色

- **主選單系統 (Menu System)**：
  - 可互動的選單介面。
  - 支援動態切換遊戲模式。
  - **戰績顯示**：在菜單上即時顯示上一局的比賽結果（分數與獲勝者）。
- **雙重遊戲模式**：
  - **CLASSIC (經典模式)**：單人遊玩，挑戰最高分。
  - **DUAL (雙人對戰)**：本地 1v1 對戰，兩隻蛇互不碰撞，比拚生存能力。
- **勝負判定機制**：
  - 在雙人模式中，若玩家撞牆或撞到自己，系統將判定對手直接獲勝。
- **模組化架構**：程式碼結構清晰，易於維護與擴充 (Menu, Game, Snake, Settings 分離)。

## 🎮 操作說明

| 功能 | 玩家 1 (綠色) | 玩家 2 (黃色) | 菜單 / 系統 |
| :--- | :---: | :---: | :---: |
| **上** | `↑` (上鍵) | `W` | - |
| **下** | `↓` (下鍵) | `S` | - |
| **左** | `←` (左鍵) | `A` | - |
| **右** | `→` (右鍵) | `D` | - |
| **切換模式** | - | - | `M` |
| **開始遊戲** | - | - | `ENTER` |

## 📂 檔案結構說明

- `menu.py`: 程式進入點 (Entry Point)，負責繪製菜單與接收回傳戰績。
- `main.py`: 包含 `run_game()` 函數，管理遊戲運行時的迴圈。
- `game.py`: 處理遊戲核心邏輯（規則、碰撞、勝負）。
- `snake.py`: 定義 `Snake` 物件（移動軌跡、身體增長）。
- `settings.py`: 設定檔（顏色、視窗大小、FPS 速度）。
